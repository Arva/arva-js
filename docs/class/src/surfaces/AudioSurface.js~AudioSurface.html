    AudioSurface

    Extends:

    famous/core/Surface.js~FamousSurface → AudioSurface

    A class for making html5 audio elements targets inside an Arva application, containing a renderable audio fragment. Like an HTML div, it can accept internal markup, properties, classes, and handle events.
